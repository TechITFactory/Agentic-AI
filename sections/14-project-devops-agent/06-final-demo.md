# Final Demo and Interview Talking Points

## Why this module

Package the project for showcasing and interviews.

## Outcomes

- ✅ Demo script with a safe task run
- ✅ Architecture and guardrails summary
- ✅ Risks/next steps articulated

**Time**: ~20-30 minutes

**Tasks**:

1) Record or script a demo scenario
2) Summarize architecture (tools, loop, guardrails)
3) List limitations and planned improvements

**Deliverable**:

- Demo script/notes and one-page summary

**Checkpoint**:

Can you explain how you prevent unsafe commands during the demo?
