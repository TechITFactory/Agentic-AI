# Agent Requirements and Tool List

## Why this module

Define the agent’s scope and the tools it is allowed to use before building.

## Outcomes

- ✅ Clear problem statement and success criteria
- ✅ Tool allowlist (log analyzer, k8s helper, YAML generator)
- ✅ Safety constraints noted

**Time**: ~20 minutes

**Tasks**:

1) Document use cases and non-goals
2) List tools with inputs/outputs and limits
3) Define budgets (time/steps/cost)

**Deliverable**:

- Requirements doc and tool list

**Checkpoint**:

Can a reviewer approve scope and tools without further clarification?
