# System Design: Ops Agent

## Why this lesson

Show you can design an agentic system with safety and observability.

## Outcomes

- ✅ Architecture for an ops agent with tools and guardrails
- ✅ Logging/monitoring plan
- ✅ Failure modes considered

**Time**: ~30 minutes

**Tasks**:

1) Sketch agent architecture (API, tools, guardrails, memory)
2) Address safety, budgets, observability
3) Prepare trade-offs

## Deliverable

- One-page agent design diagram/notes

## Checkpoint

Can you explain how the agent prevents unsafe commands and logs actions?
