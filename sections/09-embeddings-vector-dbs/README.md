# Section 9: Embeddings & Vector Databases (Foundation for RAG)

## Overview

Learn how to represent text as vectors, store them efficiently, and run similarity search. You will experiment with chunking strategies, metadata filters, and build two mini labs to cement the concepts.

## What You'll Learn

1. **Embeddings 101**: What they are and why they matter
2. **Similarity Search**: Distance metrics and ANN basics
3. **Chunking Strategies**: How chunking drives retrieval quality
4. **Metadata Filtering**: Source, date, and type filters
5. **Mini Lab**: Generate embeddings for documents
6. **Mini Lab**: Vector search with a simple dataset

1. [What Are Embeddings?](./01-what-are-embeddings.md)

## Duration

**Estimated Time**: 8-10 hours

## Prerequisites

- Section 8 completed (Intro to LLMs)
- Comfortable with Python and JSON

## What You'll Build

- A small embedding pipeline that chunks and tags documents
- A vector search demo with metadata-aware filtering
