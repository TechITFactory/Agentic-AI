# Answer with Citations

## Why this module

Return trustworthy responses grounded in retrieved sources.

## Outcomes

- ✅ Prompt template that enforces citations
- ✅ Structured responses (JSON) with source IDs/snippets
- ✅ Handle no-answer cases

**Time**: ~30 minutes

**Tasks
1) Add citations to prompts
2) Return JSON with answer + sources
3) Handle insufficient evidence gracefully

**Deliverable
- Response samples showing citations

**Checkpoint
Do all answers include real source IDs, not fabricated ones?
