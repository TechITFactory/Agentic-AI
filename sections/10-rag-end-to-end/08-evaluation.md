# Evaluation: Golden Dataset and recall@k

## Why this lesson
Measure retrieval quality with a small labeled set instead of vibes.

## Outcomes
- ✅ Build a tiny golden set (query → relevant docs)
- ✅ Compute recall@k and track over changes
- ✅ Tie evals to chunk/query tweaks

**Time**: ~25 minutes

**Prerequisites**: Retrieval improvements

## Agenda
- Golden set creation
- recall@k calculation
- Using evals to guide tweaks

## Hands-on
- Create 10-20 query→relevant IDs pairs
- Compute recall@k for two retrieval configs

## Deliverable
- Eval notebook with recall@k results

## Checkpoint
Can you show which config improves recall@5 without guessing?
