# Citations and Grounding

## Why this lesson

Ensure outputs are anchored to sources to build trust.

## Outcomes

- ✅ Include citations in responses
- ✅ Return source snippets/URLs
- ✅ Avoid fabricating references

**Time**: ~20 minutes

**Prerequisites**: Simple RAG pipeline

## Agenda

- Prompting for citations
- Structuring outputs with source IDs
- Handling missing evidence

## Hands-on

- Modify prompt to include citations
- Return structured JSON with answer + sources

## Deliverable

- Response examples with citations included

## Checkpoint

Do all answers reference actual retrieved chunks?
