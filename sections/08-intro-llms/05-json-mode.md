# JSON Mode and Schema Validation

## Why this lesson
Get reliable machine-readable outputs from LLMs.

## Outcomes
- ✅ Request JSON mode (or equivalents) from providers
- ✅ Validate responses against a schema
- ✅ Handle retries/failures gracefully

**Time**: ~20 minutes

**Prerequisites**: Prompt patterns

## Agenda
- Provider-specific JSON settings
- Defining schemas and validating outputs
- Retry/repair strategies

## Hands-on
- Call an LLM with JSON mode; validate output
- Add a simple retry on validation failure

## Deliverable
- Code snippet showing JSON request + validation

## Checkpoint
Can your code reject malformed outputs and retry safely?
